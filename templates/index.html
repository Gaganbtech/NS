📚 Quick Reference

RSA (Classical)

Traditional TLS using RSA key exchange. Fast and widely supported, but vulnerable to Shor's algorithm on quantum computers.

MLKEM768 (PQC)

NIST-standardized Module-Lattice-based Key Encapsulation Mechanism. Provides quantum-resistant security based on hard lattice problems.

Hybrid Mode

Combines X25519 (ECDH) with MLKEM768. Provides security against both classical and quantum attacks. Recommended for production.
