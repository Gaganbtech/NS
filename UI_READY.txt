🚀 HOW TO RUN:
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Install Flask
    pip3 install flask

STEP 2: Start TLS Server (Terminal 1)
    cd scripts
    ./start_server.sh

STEP 3: Start Flask Dashboard (Terminal 2)
    python3 app.py

STEP 4: Open Browser
    http://localhost:5000
